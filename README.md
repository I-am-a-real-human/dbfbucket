# dbfbucket

Rust library for reading and writing DBF files with python bindings.

## ❗ Caveat Emptor

Accordingly, I have no clue that it will consistently work, I cannot
guarantee that it won't send your bank details to North Korea, it may

## Installation

Install via pip or uv:

```sh
pip install dbfbucket
```

## Usage

**read a DBF file**

```python
from dbfbucket import DBF

db = DBF.read("path/to/file.dbf")
```

**write out data**

```python
from dbfbucket import DBF

db = DBF.read("path/to/file.dbf")

db.to_csv("output.csv")
db.to_parquet("output.parquet")
```

**Summarise columns**

```python
from dbfbucket import DBF

db = DBF.read("path/to/file.dbf")

for col in db.field_names():
    print(f"Sum of {col} is {db.get_column(col).sum()}") # mean/min/max/count

```
